This package implements a node for baseline removal, specifically catered to the use of mass spectrometry datasets, but can be used for other data, it specifically removes the baseline from each row in the incoming data.

The BaselineModel is dependent on msk.util.preprocess.Baseline which stores all the logic for 4 methods of baseline removal.

The BaselineDialog provides a very basic GUI for the user to select only the method of baseline removal, and to choose the size of the window in which we calculate the baseline. The class extends DefaultNodeSettingsPane which makes this easy and quick to code.
