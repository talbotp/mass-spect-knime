This package implements three types of Normalization in the domain of mass spectrometry data as a KNIME node.

The NormalizationModel contains the KNIME implementation of the Normalization logic. This class is dependent on msk.util.preprocess.Normalizer which stores all logic.

The NormalizationDialog is a basic GUI which allows the user to choose what method of normalization they are going to use. It extends DefaultNodeSettingsPane as it is easy to use and quick to code.
