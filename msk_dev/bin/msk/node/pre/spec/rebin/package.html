This package implements a KNIME node for Rebinning Mass spectrometry datasets.

The RebinnerModel stores all KNIME logic, and an interface between the logic and KNIME. It is dependent on msk.util.preprocess.Rebinner which stores the logic. The logic is based off of Algorithm 2 in Alan Races Spectral Analysis paper.

The RebinnerDialog is a simple GUI allowing the user to specify the minimum and maximum m/z values, and the size of the bins to be used. The class extends DefaultNodeSettingsPane for ease and speed of coding.
